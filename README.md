# C&E - Tienda de Trabajo Artesanal en Cuero Nacional

Bienvenido al repositorio de **C&E**, una landing page exclusiva y elegante diseñada para exhibir y gestionar pedidos de artículos de cuero artesanales de alta calidad con sello nacional. 

Este proyecto ha sido desarrollado como una solución web moderna, fluida y completamente adaptada a la identidad visual de una marca premium.

---

## 🌟 Características del Proyecto

* **Diseño Exclusivo y Elegante:** Interfaz limpia con una paleta de colores sofisticada (tonos crema, café y negro) que evoca la textura y calidad del cuero.
* **Navegación Fluida (Single Page Application / Smooth Scroll):** Menú de navegación interactivo que permite desplazarse suavemente entre las secciones de *Inicio*, *Nosotros*, *Productos* y *Contacto*.
* **Catálogo Dinámico e Interactivo:** * Sección principal de colecciones (*Cinturones, Carteras, Billeteras y Accesorios*).
    * Vistas detalladas para cada categoría mediante interacciones que muestran la disponibilidad de stock y especificaciones de cada producto de manera clara.
* **Sección de Contacto y Canales de Venta:** Integración directa y visible a redes sociales oficiales y WhatsApp para la gestión personalizada de pedidos exclusivos.
* **Totalmente Responsivo:** Adaptado para ofrecer la mejor experiencia visual tanto en computadoras de escritorio como en dispositivos móviles.

---

## 🛠️ Tecnologías Utilizadas

Para la construcción de este sitio web se utilizaron tecnologías estándar de la industria web, priorizando la optimización del rendimiento y un código limpio:

* **HTML5:** Estructuración semántica del contenido para mejorar el SEO y la accesibilidad.
* **CSS3:** Estilizado personalizado, uso de Flexbox/Grid para el diseño responsivo, transiciones suaves y efectos visuales modernos.
* **JavaScript (Vanilla JS):** Lógica interactiva para la navegación entre secciones, control del flujo del catálogo de productos y dinamismo de la interfaz sin depender de librerías pesadas.

---

## 📁 Estructura del Sitio

1.  **Hero Section (Inicio):** Banner principal con una imagen de alta calidad, eslogan de la marca ("Elegancia perdurable hecha a mano") y botón de llamada a la acción (*Explorar Catálogo*).
2.  **Nosotros:** Breve introducción que destaca la dedicación absoluta detras de cada pieza y su idoneidad para obsequios.
3.  **Productos (Nuestras Colecciones):** Módulos interactivos que desglosan el catálogo en categorías con galerías de fotos detalladas.
4.  **Contacto / Pie de Página:** Información clave sobre horarios de atención, despacho a todo Chile, medios de pago y enlaces directos a canales de comunicación.
